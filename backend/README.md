Backend (Django + DRF)

This folder will house the Django project and apps.

Next steps to finish backend scaffold:
- `django-admin startproject app .` inside `backend` directory
- Create an app (e.g., `content`) and implement models for Portfolio, Blog, Testimonial
- Configure `settings.py` to use REST framework, simplejwt, CORS, django-storages
- Implement upload endpoints and file validation
- Implement sitemap and robots endpoints

Use the provided `requirements.txt` to install dependencies.
